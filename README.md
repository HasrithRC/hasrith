# TravelNest 🌍✈️

A comprehensive travel planning web application that helps you discover, plan, and organize your next adventure.

## Features

### 🏠 **Home Page**
- Hero section with inspiring travel imagery
- Featured destinations showcase
- Quick search functionality for discovering locations
- Navigation to all key features

### 🗺️ **Destination Explorer**
- Browse all available travel destinations
- Advanced search and filtering options
- Detailed destination information
- Gallery view with destination images

### 💰 **Budget Planner**
- Create and manage travel budgets
- Track expenses by category
- Calculate total trip costs
- Real-time budget updates

### 🎲 **Random Trip Generator**
- Generate random destination suggestions
- Mood-based trip recommendations
- Discover unexpected adventures
- Perfect for spontaneous travelers

### 😊 **Travel Moods**
- Select your travel mood and preferences
- Get personalized destination recommendations
- Ambient soundscapes for each mood (beach, city, forest, rain)
- Customize your travel experience

### 💬 **Feedback & Support**
- Contact form for user inquiries
- Report issues or share suggestions
- Customer support communication
- Email feedback submission

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server or installation required

### Installation

1. Clone or download the project:
```bash
git clone https://github.com/yourusername/TravelNest.git
cd TravelNest
```

2. Open in browser:
```bash
# Simply open index.html in your web browser
```

Or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server
```

3. Navigate to `http://localhost:8000` in your browser
